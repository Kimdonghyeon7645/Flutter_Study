# 3-1 - flutter 앱을 웹으로 변환

사실 Flutter 로, 데스크탑(PC)의 .exe 파일을 만들수도 있고, 웹 어플리케이션도 가능하다고 한다.

이번 장에서는 웹 애플리케이션으로 포팅해보는 작업을 해볼 것이다.  
지난시간에 만들었던 앱을 이용해서다.
